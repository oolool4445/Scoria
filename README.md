# Scoria Project (C#)

A simple C# console application themed around scoria, a volcanic rock.

## Features
- Object-oriented design
- Multiple classes (Scoria, Volcano, Lava, etc.)
- Educational theme about volcanic processes

## How to Run
1. Open in Visual Studio or VS Code
2. Compile and run Program.cs

## Example Output
Welcome to the Scoria Project!
Volcanic Scoria is a porous volcanic rock with a Dark Red color.
Mount Ember is erupting and releasing lava!
